# AI RAG Mock Interview Agent

An AI-powered mock interview evaluation system that uses Sentence Transformers and Cosine Similarity to assess candidate answers based on semantic similarity.

## 🚀 Features

- RAG-inspired evaluation approach
- Semantic similarity scoring using all-MiniLM-L6-v2
- Real-time answer evaluation
- Detailed performance feedback
- Reference answer comparison

## 🧠 Tech Stack

- Python
- Sentence Transformers
- Scikit-learn
- Cosine Similarity
- JSON Knowledge Base
